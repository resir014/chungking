# chungking

> @resir014's personal design system.

## Getting started

A [Node.js](https://nodejs.org/en/) >= 14.0.0 with yarn is recommended.

```bash
# install dependencies
yarn

# build all packages
yarn build

# run all tests
yarn test

# run tests in-module
yarn workspace [package-name] run test
```
